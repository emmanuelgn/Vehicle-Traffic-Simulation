# Vehicle-Traffic-Simulation

This project is a basic traffic simulation in a city, featuring cars, traffic lights, and roads.

## How to Compile

To compile the project, you can use CMake:

```bash
mkdir build
cd build
cmake ..
make
```

## How to Run

After compilation, run the program with:

```bash
./TrafficSimulation
```
